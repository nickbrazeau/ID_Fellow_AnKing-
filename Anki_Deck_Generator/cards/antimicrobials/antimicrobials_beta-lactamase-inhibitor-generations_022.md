## Question

Compare **first-generation** (clavulanate, sulbactam, tazobactam) vs **new-generation** (avibactam, vaborbactam, relebactam) β-lactamase inhibitors. What β-lactamases does each class inhibit?

## Answer

| Feature | First-Gen (CST) | New-Gen (AVR) |
|---------|-----------------|---------------|
| **Agents** | **C**lavulanate, **S**ulbactam, **T**azobactam | **A**vibactam, **V**aborbactam, **R**elebactam |
| **Structure** | β-lactam ring | Non-β-lactam (diazabicyclooctane for AVR) |
| **Mechanism** | Suicide inhibitors (irreversible) | Competitive/reversible binding |
| **Class A inhibition** | TEM, SHV (NOT ESBLs reliably) | YES - including KPC carbapenemases |
| **Class C (AmpC)** | NO | YES (avibactam, relebactam) |
| **Class D (OXA)** | NO | Avibactam: some OXA-48 |
| **Class B (MBL)** | NO | NO |

### **Detailed Coverage:**

**First-Generation (Clavulanate > Tazobactam > Sulbactam):**
- Inhibit plasmid-mediated penicillinases (TEM-1, SHV-1)
- Do NOT inhibit AmpC or carbapenemases
- Clinically unreliable vs ESBL producers (despite in vitro activity)
- **Potency:** Clavulanate = Tazobactam >> Sulbactam (clavulanate 60× > sulbactam vs TEM-1)

**New-Generation:**
- **Avibactam:** Class A + C + some D (NOT MBLs)
- **Vaborbactam:** Class A only (KPC) - NOT OXA or MBLs
- **Relebactam:** Class A + C (NOT carbapenemases)

## Key Points

### **Clinical Combinations:**
- Ceftazidime-**avibactam**: KPC-producing CRE, *P. aeruginosa* with AmpC
- Meropenem-**vaborbactam**: KPC-producing CRE (Gram-neg only)
- Imipenem-**relebactam**: AmpC-producing organisms, NOT carbapenemases
- Cefiderocol: "Trojan horse" siderophore - active vs MBLs (different mechanism)

### **Gaps in Coverage:**
- **NO inhibitor** covers **MBLs** (NDM, VIM, IMP) effectively
- Aztreonam + ceftazidime-avibactam combination needed for MBL producers
