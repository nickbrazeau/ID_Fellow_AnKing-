## Question

How is osteomyelitis diagnosed and treated? Use the **"Bone Biopsy → Prolonged Antibiotics (6-8 weeks) ± Surgery"** principle.

## Answer

### **Diagnosis:**

**Gold Standard:** **Bone biopsy** (culture + histopathology)
- Most accurate
- Guides targeted therapy
- Essential if hardware present

**Imaging:**
- **MRI** (most sensitive, 90-100%)
- **X-ray** (insensitive early, abnormal after 10-14 days)
- **Nuclear medicine** (bone scan, WBC scan) if MRI contraindicated

### **Empiric Therapy (by Scenario):**

**Hematogenous (children/adults without hardware):**
- **Nafcillin/oxacillin** 2g IV q4h (MSSA) OR
- **Vancomycin** 15-20 mg/kg IV q12h (if MRSA risk)

**Contiguous (post-trauma, surgery):**
- **Vancomycin + cefepime** (or pip-tazo)
- Cover MRSA + Gram-negatives

**Diabetic Foot:**
- **Vancomycin + pip-tazo** (or ceftriaxone + metronidazole)
- Cover MRSA + Gram-negatives + anaerobes

**Hardware-Associated:**
- **Vancomycin + cefepime + rifampin**
- Rifampin for biofilm penetration

### **Duration:**
- **6-8 weeks** IV/PO therapy
- **Longer** if hardware retained, immunocompromised, or inadequate debridement

## Key Points

### **Rifampin for Hardware-Associated:**
- **Penetrates biofilms** (essential for prosthetic joint infections)
- **Always combine** with another agent (rapid resistance if alone)
- Typical: **Rifampin + fluoroquinolone** or **rifampin + minocycline**

### **Surgery Indications:**
- **Necrotic bone** (requires debridement)
- **Abscess/sequestrum**
- **Hardware removal** if infected and removable
- **Failed medical therapy**

### **Clinical Pearls:**
- **Bone biopsy** before antibiotics if possible
- **6-8 weeks** minimum duration
- **Rifampin** essential for hardware-associated (biofilm penetration)
- **MRI** best imaging modality
- Consider **oral step-down** after 2-4 weeks IV if responding
