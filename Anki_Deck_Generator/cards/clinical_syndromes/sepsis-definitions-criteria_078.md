## Question

What are the Sepsis-3 definitions for sepsis and septic shock? Use the **"SOFA ≥2 + SHOCK = MAP<65 + Lactate>2"** mnemonic.

## Answer

### **"SOFA ≥2 + SHOCK = MAP<65 + Lactate>2" Mnemonic:**

**SOFA ≥2:**
- **S**epsis = infection + SOFA ≥2
- **O**rgan dysfunction (≥2 point increase)
- **F**rom baseline
- **A**cute change

**SHOCK:**
- **S**eptic shock = sepsis + hypotension + lactate
- **H**ypotension requiring vasopressors
- **O**utcome worse (40% mortality)
- **C**ell ular hypoperfusion (lactate >2)
- **K**ey: MAP ≥65 goal despite fluids

**MAP<65 + Lactate>2:**
- **M**AP must be maintained ≥65 with vasopressors
- **A**fter adequate fluid resuscitation
- **P**ersistent hypotension
- **Lactate >2 mmol/L** (despite fluids)

### **Sepsis-3 Definitions (2016):**

**1. Sepsis:**
- **Infection** (documented or suspected) PLUS
- **Acute organ dysfunction:** SOFA score ≥2 points from baseline

**2. Septic Shock:**
- **Sepsis** PLUS **both** of:
  - Vasopressor requirement to maintain **MAP ≥65 mmHg**
  - **Lactate >2 mmol/L** (18 mg/dL)
  - Despite adequate fluid resuscitation

**Mortality:**
- Sepsis: ~10%
- Septic shock: ~40%

### **qSOFA (Quick SOFA - Bedside Screening):**

**Purpose:** Rapid bedside tool to identify patients at risk for poor outcomes

**Criteria (≥2 positive = high risk):**
1. **R**espiratory rate ≥22/min
2. **A**ltered mentation (GCS <15)
3. **S**ystolic BP ≤100 mmHg

**≥2 qSOFA criteria → Higher mortality risk (consider full SOFA, escalate care)**

## Key Points

### **SOFA Score Components (0-4 points each, max 24):**

| System | Dysfunction Marker |
|--------|-------------------|
| **Respiration** | PaO2/FiO2 ratio |
| **Coagulation** | Platelets |
| **Liver** | Bilirubin |
| **Cardiovascular** | MAP, vasopressor requirement |
| **CNS** | Glasgow Coma Scale |
| **Renal** | Creatinine or urine output |

**Sepsis = SOFA ≥2 points increase from baseline**

### **Key Changes from Sepsis-2 to Sepsis-3:**

| Feature | Sepsis-2 (Old) | Sepsis-3 (2016) |
|---------|----------------|-----------------|
| **SIRS criteria** | Required | **Removed** (not specific) |
| **Definition** | SIRS + infection | **SOFA ≥2** + infection |
| **Severe sepsis** | Separate category | **Eliminated** (just "sepsis" now) |
| **Septic shock** | SBP <90 or MAP <65 | **MAP <65 + lactate >2 despite fluids** |

### **Clinical Pearls:**
- **SIRS no longer used** (too sensitive, not specific)
- **qSOFA ≥2** = bedside red flag (get full SOFA, escalate care)
- **Lactate >2** despite fluids = septic shock (poor prognosis)
- **MAP goal ≥65** with vasopressors
- Septic shock mortality ~**40%**
